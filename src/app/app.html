Angular Tabs Challenge

Implement the logic in TabsComponent and TabComponent to make the following example work. The tabs should switch content when clicked.

Welcome to the Challenge!

Your task is to make these tabs functional.

  • The header should automatically show titles from the app-tab children.
  • Clicking a title should show the corresponding content.
  • Only the active tab's content should be rendered/visible.
