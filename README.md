# Angular Tabs Challenge

Welcome to the Angular Tabs Challenge! Your goal is to build a custom tab component from scratch that behaves similarly to Angular Material Tabs. 

## Requirements

1. **Create a component that behaves like Angular Material Tabs:** 
   You'll be working with `<app-tabs>` (the container) and `<app-tab>` (the individual tab).
2. **Render Custom Content:**
   Each tab should be able to render whatever HTML content the user wants to display.
3. **Define Tab Titles:**
   The user must be able to define a `title` for each tab.
4. **Interactive:**
   Clicking on a tab title should switch the visible content to the selected tab. Only one tab's content should be visible at a time.
5. **From Scratch:**
   Do not use any UI libraries like Angular Material for the core functionality. Use standard Angular features.

## Getting Started

1. The project has an initial skeleton set up. Check `src/app/tabs/tabs.component.ts` and `src/app/tabs/tab/tab.component.ts`.
2. The `src/app/app.html` file contains an example of how your `<app-tabs>` and `<app-tab>` should be used. Currently, they don't do anything! 
3. Implement the logic to make the tabs functional.

Good luck!
